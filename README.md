A comprehensive, multi-page Power BI dashboard designed to analyze global population trends, economic health (GNP), and cultural metrics. This project transforms raw global data into actionable insights regarding life expectancy, urbanization, and linguistic diversity.


## üöÄ Key Features
* **World Overview:** Executive summary of global population (6bn+), GNP ($29.35M), and average life expectancy (66.48).

* **Economic Analysis:** Breakdown of Total GNP by Government Form (e.g., Federal Republic vs. Constitutional Monarchy).

* **Interactive Navigation:** Custom-built landing page with seamless navigation between reports.
